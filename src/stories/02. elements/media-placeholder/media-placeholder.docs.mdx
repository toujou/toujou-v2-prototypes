# Media placeholder

The `toujou-media-placeholder` is shown whenever we can't render some kind of media

## Markup example
```html
    <toujou-media-placeholder class="media-placeholder" media-type="image"></toujou-media-placeholder>
```

## Attributes

### media-type
With the `media-type` attribute we can set what type of media the element is replacing. It will show the corresponding icon

Accepted values are:
- `image`
- `video`
- `audio`
