# Portfolio gallery

- The portfolio gallery render a slider on mobile and an "image grid" on desktop devices.
- On mobile it shows a "counter" element
- On desktop the button is shown only if the maximum number of images is achieved

## The slots

### slider slot <toujou-chip class="chip" chip-bg-color="warning">required</toujou-chip>
The slider element must have the `slot="slider"` attribute

### button slot <toujou-chip class="chip" chip-bg-color="warning">required</toujou-chip>
The button element must have the `slot="button"` attribute

## Attributes

### number-of-items <toujou-chip class="chip" chip-bg-color="warning">required</toujou-chip>
The `number-of-items` attribute on the portfolio element defines how many images we will show on desktop devices and should reflect the number of images on the slider.

Valid values are `1`, `2`, `3`, `4`, `5` and `max`. `max` should be added when the number of images is greater than 5

### slider-type <toujou-chip class="chip" chip-bg-color="warning">required</toujou-chip>
The slider element inside the portfolio element must have the `slider-type="slider"` attribute

### is-header <toujou-chip class="chip" chip-bg-color="warning">required</toujou-chip>
The `is-header` attribute must be added to the portfolio gallery element when it is displayed inside the header section of the page.
