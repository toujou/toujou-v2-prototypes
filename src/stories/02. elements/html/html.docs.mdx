# HTML

The HTML element allows users to add their own custom html to the page. You can add it to the page in the BE.

## Markup example
```html
<toujou-html-container class="html-container" fullwidth>
    <!-- Here goes the custom html -->
</toujou-html-container>
```

## Attributes

### fullwidth
There are two different HTML content elements on the back:
    - content width
    - fullwidth
If the users select the fullwidth HTML content element, that it must have the `fullwidth` attribute
