# spinner

The `<toujou-spinner>` element renders a spinner on the page so show that something is loading.

<toujou-spinner class="spinner"></toujou-spinner>

## Markup example
```html
    <toujou-spinner class="spinner"></toujou-spinner>
```

## Attributes
There are some attributes that allow us to customize the spinner element.


### centered-on-page
By default the `<toujou-spinner>` element is rendered inline. If we need to show it centered on the page and on top of all the content we can give it the `centered-on-page` attribute.

Example:
```html
    <toujou-spinner class="spinner" centered-on-page></toujou-spinner>
```
