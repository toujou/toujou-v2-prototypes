# &lt;toujou-button&gt;
The `<toujou-icon>` is a simple custom element that allows us to render an icon on the page.

| We also support the old (deprecated) button classes, like `button--shadow`. Refer to the "Button Old" story

## Example
<button is="toujou-button" class="button" button-variant="primary" button-type="default" button-size="normal">Hello world</button>


## Markup example
```html
<button is="toujou-button" class="button" button-variant="primary" button-type="default" button-size="normal">Hello world</button>
```

## Attributes
There are some attributes that allow us to customize the button


### button variant <toujou-chip class="chip" chip-bg-color="warning">required</toujou-chip>
By setting the `button-variant` attribute we can choose between primary, secondary or font color buttons

| Name | Attribute | Example |
|---|---|:---:|
| primary | `button-variant="primary"`  | `<button is="toujou-button" class="button" button-variant="primary" button-type="default" button-size="normal">Hello world</button>` |
| secondary | `button-variant="secondary"`  | `<button is="toujou-button" class="button" button-variant="secondary" button-type="default" button-size="normal">Hello world</button>` |
| font | `button-variant="font"`  | `<button is="toujou-button" class="button" button-variant="font" button-type="default" button-size="normal">Hello world</button>` |


### button type <toujou-chip class="chip" chip-bg-color="warning">required</toujou-chip>
By setting the `button-type` attribute we can choose between different kinds of designs for the buttons

| Name | Attribute | Example |
|---|---|:---:|
| default | `button-type="primary"`  | `<button is="toujou-button" class="button" button-variant="primary" button-type="default" button-size="normal">Hello world</button>` |
| border | `button-type="secondary"`  | `<button is="toujou-button" class="button" button-variant="primary" button-type="border" button-size="normal">Hello world</button>` |
| normal | `button-type="normal"`  | `<button is="toujou-button" class="button" button-variant="font" primary-type="ghost" button-size="normal">Hello world</button>` |
| large | `button-type="large"`  | `<button is="toujou-button" class="button" button-variant="font" primary-type="ghost" button-size="large">Hello world</button>` |


### button size <toujou-chip class="chip" chip-bg-color="warning">required</toujou-chip>
By setting the `button-size` attribute we can choose between different text / icon sizes

| Name | Attribute | Example |
|---|---|:---:|
| tiny | `button-size="tiny"`  | `<button is="toujou-button" class="button" button-variant="primary" button-type="default" button-size="tiny">Hello world</button>` |
| small | `button-size="small"`  | `<button is="toujou-button" class="button" button-variant="primary" button-type="border" button-size="small">Hello world</button>` |
| ghost | `button-size="font"`  | `<button is="toujou-button" class="button" button-variant="font" primary-type="ghost" button-size="normal">Hello world</button>` |


### button icon position
By setting the `icon position` attribute we can choose if the icon is show on the right or left side of the button's text. It must be used in combination with a `<toujou-icon>` element.

Example:
```html
<button is="toujou-button" class="button" button-variant="primary" button-type="default" button-size="normal" button-icon-position="left">
    <toujou-icon class="icon" icon-name="arrow-right"></toujou-icon> Hello world
</button>
```

| Attribute | Possible values |
|---|-----------------|
| button-icon-position | "left", "right" |


### is-expanded
By setting the `button-type` attribute we can make the button fullwidth

| Attribute | Example |
|--|:---:|
| `is-expanded`  | `<button is="toujou-button" class="button" button-variant="primary" button-type="default" button-size="normal" is-expanded>Hello world</button>` |


### disabled
By setting the `disabled` attribute we can disable the button

| Attribute | Example |
|--|:---:|
| `disabled`  | `<button is="toujou-button" class="button" button-variant="primary" button-type="default" button-size="normal" disabled>Hello world</button>` |
