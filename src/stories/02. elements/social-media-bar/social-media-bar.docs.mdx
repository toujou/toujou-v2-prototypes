# Social media bar

The social media bar element is rendered when the "Social Media Bar" content element is used.

It renders a section on the page containing a headline (optional) and links for social media accounts.

It can be displayed in the default, 'primary', 'secondary' or 'inverted' designs, which are selected on the BE.

## Markup example:
```html
<toujou-social-media-bar class="social-media-bar" element-design="default">
    <h3 class="social-media-bar__headline">Get in touch</h3>
    <div class="social-media-bar__links">
        <a href="#" class="social-media-bar__link" aria-label="Our facebook account">
            <toujou-icon class="icon" icon-name="sm-facebook" icon-size="xxl" icon-color="primary" aria-hidden></toujou-icon>
        </a>
        <a href="#" class="social-media-bar__link" aria-label="Our twitter account">
            <toujou-icon class="icon" icon-name="sm-twitter" icon-size="xxl" icon-color="primary" aria-hidden></toujou-icon>
        </a>
    </div>
</toujou-social-media-bar>
```

## Attribute
The `no-headline` attribute should be added to the element if relevant.
```html
<toujou-social-media-bar class="social-media-bar" element-design="default" no-headline>
    <div class="social-media-bar__links">
        <a href="#" class="social-media-bar__link" aria-label="Our facebook account">
            <toujou-icon class="icon" icon-name="sm-facebook" icon-size="xxl" icon-color="primary" aria-hidden></toujou-icon>
        </a>
    </div>
</toujou-social-media-bar>
```
