# Topbar

The topbar is a complex element, which render a topbar possibly with a main navigation, service navigation, logo and burger button.

The topbar element is composed by different elements, which also have they own docs:
- [main navigation](#main-navigation)
- [service navigation](#service-navigation)
- [burger button](#burger-button)

## Options
The topbar isn't a content element. It is automatically generate according to the page's content and settings.

There are some settings that change the way the topbar looks and behaves:

### Navigation theme
The navigation theme can be set on the "toujou theme" module, on the "Theme" tab. We can set the "Enable light navigation colors" option either to `dark navigation` or `light navigation`

This will in turn set the correct attribute (`navigation-theme`) value on the topbar element, and therefore giving the topbar the correct styles.

Example: `<toujou-topbar id="topbar" class="topbar" navigation-theme="dark" logo-size="medium">...<toujou-topbar>`

### Page title type
On the "toujou theme" module, on the "Theme" tab we can choose between `Site title` and `Top bar logo` on the "Mobile top bar content option.

This will render either the logo or the site title on the topbar (not only on mobile, like the option name says)

Example:
```html
<toujou-topbar id="topbar" class="topbar" navigation-theme="light" logo-size="medium">
    <a href="#" class="topbar__logo-link">
        <span class="topbar__title">Nice title</span>
    </a>
    ...
</toujou-topbar>
```

### Logo size
The logo size can be defined on the "toujou theme" module "Logo" tab. For the "Logo size" we can choose between `small`, `medium` and `large`.

This will affect the height of the whole topbar, not only the logo size.

Example: `<toujou-topbar id="topbar" class="topbar" navigation-theme="light" logo-size="small">...</toujou-topbar>`

## Attributes

### navigation-theme <toujou-chip class="chip" chip-bg-color="warning">required</toujou-chip>
Change between light and dark theme. Valid values are `light` and `dark`

Example: `<toujou-topbar id="topbar" class="topbar" navigation-theme="dark" logo-size="medium"></toujou-topbar>`

### logo-size <toujou-chip class="chip" chip-bg-color="warning">required</toujou-chip>
Change between the different logo sizes, which affects the topbar's height. Valie values are `small`, `medium` and `large`

Example: `<toujou-topbar id="topbar" class="topbar" navigation-theme="dark" logo-size="large"></toujou-topbar>`

## Events
The `<toujou-topbar>` element dispatches and reacts to some custom events.

### toujou-topbar-breakpoint-change
Whenever the window size changes and the breakpoint between mobile and desktop is crossed, the topbar dispatches a custom `toujou-topbar-breakpoint-change` event.

### toujou-burger-button-click
The topbar element reacts to the `toujou-burger-button-click` by setting / removing the open navigation attribute on itself according to the received state
