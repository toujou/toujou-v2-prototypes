# &lt;toujou-form&gt;
The toujou form element (with the class `toujou-form`) render form and form elements on the page. It can be created in a couple different ways, but most probably it will be created using the "Forms" module on the BE and then placed on a page using the "Mail Form" plugin.

> ⚠️ These are the form elements still using the old markup

Until we have time to complete this documentation, please refer to the stories to see the correct markup for the form elements
