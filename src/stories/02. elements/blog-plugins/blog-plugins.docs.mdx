# Blog plugin - authors

The blog plugin is available as a content element on the BE. It can show:
- Blog: Archive
- Blog: Authors
- Blog: Comment Form
- Blog: Comments
- Blog: Footer
- Blog: Header
- Blog: Latest Posts
- Blog: List by author
- Blog: List by category
- Blog: List by tags
- Blog: List of posts
- Blog: Metadata
- Blog: Related posts
- Blog: Sidebar

Each of these plugin types renders its own template / partial
