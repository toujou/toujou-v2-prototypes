# &lt;toujou-card&gt;
The `<toujou-card>` is a simple element that renders an card on the page.

These cards are rendered when we use a "Selected pages as card collection" or "Filtered pages as card collection" backend element.

Depending on the page we chose on the backend element, the cards may display different content (for instance the blog cards show the post date and author and the trip cards show trip info and price)

| ⚠️ The `<toujou-card>` element must be rendered inside a `<card-collection>` and `<card-collection__item>` elements |
|----------------------------------------------------------------------------------------------|



Markup Example:
```html
<ul class="card-collection">
    <li class="card-collection__item">
        <toujou-card class="toujou-card" card-type="default" card-design="default">
            <div class="toujou-card__top">
                <figure class="toujou-card__figure">
                    <img src="https://picsum.photos/640" alt="nice image" class="toujou-card__image">
                </figure>
            </div>
            <div class="toujou-card__bottom">
                <h3 class="toujou-card__title">Eine etwas längere Headline über zwei Zeilen</h3>
                <p class="toujou-card__subtitles">
                    <span class="toujou-card__subtitle">Dies ist ein Untertitel</span>
                </p>
                <p class="toujou-card__abstract">Wie Sie ein Kontaktformular (und andere Onlineformulare) erstellen, worauf aus DSGVO-Sicht und für den Spamschutz zu achten ist: Das erläutern wir in diesem Blogartikel.</p>
                <a href="#" class="button toujou-card__button" button-variant="primary" button-type="ghost" button-size="normal" button-icon-position="left">
                    <toujou-icon class="icon button__icon" icon-name="arrow-right"></toujou-icon>
                    zur Beschreibung
                </a>
            </div>
        </toujou-card>
    </li>
</ul>
```

## Attributes
There are a couple of attribute that allow us to customize the `<toujou-card>` element


### card type
The `card-type` attribute tells us what kind of page the card is rendering. It may be used by our JS or CSS to target cards of a certain page type

### card design
the `card-design` attribute reflects the "Element design" option chosen in the backend. It is used to change the style if the cards, so we can make a card standout, for instance by giving it a "primary" design
