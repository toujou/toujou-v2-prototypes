# Headlines

Different headline elements that can be used

## Markup example
```html
    <h2 type-alignment="center">This is a headline</h2>
```

## Customization

### Alignment
We can use the `type-alignment` attribute to align the text as we need.
Accepted values are:
- `left`
- `center`
- `right`

### Type color
We can use the `type-color` attribute to set the headline's color.
Accepted values are:
- `primary`
- `secondary`
- `font`

To provide backward compatibility (needed for some extensions) the following deprecated values are also valid:
- `font--primary`
- `font-secondary`
- `font--grey`

### Type size
We can use the `type-size` attribute to set the headline's size.
Accepted values are
- `font--alpha`
- `font--beta`
- `font--gamma`
- `font--delta`
- `font--epsilon`
- `font--omega`.
