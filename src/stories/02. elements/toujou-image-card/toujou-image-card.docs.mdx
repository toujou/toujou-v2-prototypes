# &lt;toujou-image-card&gt;
The `<toujou-image-card>` is a simple element that render an image card on the page (image, with a title, either as link or as a simple card)

| ⚠️ The `<toujou-image-card>` element must be rendered inside a `<toujou-image-card-grid>` element |
|---------------------------------------------------------------------------------------------------|


If there is a link available, the whole card will be linked. If no link is available, instead of an `<a>` it will be rendered as a `<toujou-image-card>` element.

Markup Example:
```html
<toujou-image-card-grid class="toujou-image-card-grid">
    <a class="toujou-image-card" href="#" image-format="square" card-variant="secondary">
        <figure class="toujou-image-card__figure">
            <img src="https://picsum.photos/400/400" alt="beautiful image" class="toujou-image-card__image" />
        </figure>
        <h3 class="toujou-image-card__title">
            <toujou-icon class="toujou-icon" icon-name="arrow-right"></toujou-icon>
            Beautiful title
        </h3>
    </a>
</toujou-image-card-grid>
```

## Attributes
There are a couple of attribute that allow us to customize the `<toujou-image-card>` element


### card variant <toujou-chip class="toujou-chip" chip-bg-color="warning">required</toujou-chip>
By setting the `card-variant` attribute we can choose between the default, primary, secondary or inverted element designs (can be set on the BE by the users)

### image format <toujou-chip class="toujou-chip" chip-bg-color="warning">required</toujou-chip>
By setting the `image-format` attribute we can choose between the "cinema" and "square" formats for the image cards image (depending on the BE element used)
