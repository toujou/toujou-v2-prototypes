# &lt;toujou-image-card&gt;
The `<toujou-icon>` is a simple custom element that allows us to render an icon on the page.


## Example
<button is="toujou-button" class="toujou-button" button-variant="primary" button-type="default" button-size="normal">Hello world</button>


## Markup example
```html
<button is="toujou-button" class="toujou-button" button-variant="primary" button-type="default" button-size="normal">Hello world</button>
```

## Attributes
There are some attributes that allow us to customize the button


### button variant <toujou-chip class="toujou-chip" chip-bg-color="warning">required</toujou-chip>
By setting the `button-variant` attribute we can choose between primary, secondary or font color buttons

| Name | Attribute | Example |
|---|---|:---:|
| primary | `button-variant="primary"`  | `<button is="toujou-button" class="toujou-button" button-variant="primary" button-type="default" button-size="normal">Hello world</button>` |
| secondary | `button-variant="secondary"`  | `<button is="toujou-button" class="toujou-button" button-variant="secondary" button-type="default" button-size="normal">Hello world</button>` |
| font | `button-variant="font"`  | `<button is="toujou-button" class="toujou-button" button-variant="font" button-type="default" button-size="normal">Hello world</button>` |
