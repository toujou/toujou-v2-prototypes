# &lt;toujou-collection-item&gt;
The `<toujou-collection-item>` is a custom element that renders a page as a card (in a card collection) or as an item (on an item collection).

These items are rendered when we use a "Selected pages as card collection" or "Filtered pages as card collection" backend element.

Depending on the page we chose on the backend element, the items may display different content (for instance the blog items show the post date and author and the trip items show trip info and price)

| ⚠️ The `<toujou-collection-item>` element must be rendered inside a `<card-collection>` and `<card-collection__item>` or inside a `ol.item-collection` and a `li.item-collection__item` elements |
|----------------------------------------------------------------------------------------------|

                </figure>

                    <a href="#" class="collection-item__category-link">
                        <toujou-chip

                            chip-icon-position="left"

                        >
                            <toujou-icon class="icon chip__icon" icon-name="bookmark-filled"></toujou-icon> ${text}
                        </toujou-chip>
                    </a>
                </div>
            </header>
            <div class="collection-item__bottom">
                <h3 class="collection-item__title">Eine etwas längere Headline über zwei Zeilen</h3>
                <p class="collection-item__subtitles">
                    <span class="collection-item__subtitle">Dies ist ein Untertitel</span>
                </p>
                <p class="collection-item__abstract">Wie Sie ein Kontaktformular (und andere Onlineformulare) erstellen, worauf aus DSGVO-Sicht und für den Spamschutz zu achten ist: Das erläutern wir in diesem Blogartikel.</p>
                <a href="#" class="button collection-item__button" button-variant="primary" button-type="ghost" button-size="normal" button-icon-position="left">
                    <toujou-icon class="icon button__icon" icon-name="arrow-right"></toujou-icon>
                    zur Beschreibung

            </div>
        </toujou-collection-item>

## Attributes
There are a couple of attribute that allow us to customize the `<toujou-card>` element


### item type <toujou-chip class="chip" chip-bg-color="warning">required</toujou-chip>
The `item-type` attribute tells us what kind of page the item is rendering. It may be used by our JS or CSS to target items of a certain page type.

Example: `item-type="default"` or `item-type="trip"`

### Element design <toujou-chip class="chip" chip-bg-color="warning">required</toujou-chip>
The `element-design` attribute reflects the "Element design" option chosen in the backend.
It is used to change the style of the items, so we can make an item standout, for instance by giving it a "primary" design

Example: `element-design="primary"` or `element-design="inverted"`.

| ℹ️ Currently the "Element design" isn't available on the "Selected pages as list" element! |
|----------------------------------------------------------------------------------------------|

### item orientation <toujou-chip class="chip" chip-bg-color="warning">required</toujou-chip>
The `item-orientation` defines if the image is rendered above the content or next to it. On mobile devices the image is always rendered above the text.

Example: `item-orientation="vertical"` (for the "cards") or `item-orientation="horizontal"` (for the "list items")

### role <toujou-chip class="chip" chip-bg-color="warning">required</toujou-chip>
By setting the `role="article"` we define the cards as a section of a page that could easily stand on its own on a page, in a document.
