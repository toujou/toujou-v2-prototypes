# &lt;toujou-details&gt;
The `<toujou-details>` renders a default html `<details>` element.


## Example
<toujou-details id="details-1" class="toujou-details">
    <h3 slot="summary" class="toujou-details__title">I am simple &lt;toujou-details&gt; element</h3>
    <toujou-icon class="toujou-icon toujou-details__chevron" icon-name="chevron-down" slot="chevron"></toujou-icon>
    <p slot="content">Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat.</p>
</toujou-details>

## Attributes
There are some attributes that allow us to customize the button


### is-open
The `is-open` attribute allows us to define if the element is open or closed when the page loads. This is set by choosing the desired option on the BE.

Example:
```html
<toujou-details id="details-1" class="toujou-details" is-open>
    ...
</toujou-details>
```
