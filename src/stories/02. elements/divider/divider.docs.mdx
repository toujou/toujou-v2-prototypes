# divider

The divider (or the `hr`) provides a thematic break block to be used between paragraph level elements, like a change of story or shift in topic within a section

## Markup example
``` html
<hr class="divider" divider-type="primary">
```

or
``` html
<hr divider-type="primary">
```

or even
``` html
<hr>
```

## Attributes

### divider-type
The divider type can be set by the user on the BE and it defines the style of the divider element.

Possible values are `background`, `primary`, `secondary` and `grey`
