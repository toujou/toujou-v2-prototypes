# Header slider element
The header slider render a slider with image and text content on the header section of the page.

It can be placed only on the "header" column on the pages.

Unlike the "header" element, the header slider element does not allow for a link and has fewer options.

## Attributes
There are a couple of attributes that allow us to customize the header element. Some attributes must be added to the `header` element, others to the `.header-content` element

### 1. image-height <toujou-chip class="chip" chip-bg-color="warning">required</toujou-chip>
Allows us to set the header's slider height in relation to the viewport.

Corresponds to the "Header Imageheight" option.

Possible values are `default`, `fullscreen`, `halfscreen` and `quarterscreen`. Default is `default` (80%).


### 2. vertical-position <toujou-chip class="chip" chip-bg-color="warning">required</toujou-chip>
Allows us to set the header's content vertical position in relation to the header element.

Corresponds to the "Vertical Position" option.

Possible values are `top`, `mid` and `bottom`. Default is `mid`;


### 3. horizontal-position <toujou-chip class="chip" chip-bg-color="warning">required</toujou-chip>
Allows us to set the header's content horizontal position in relation to the header element.

Corresponds to the "Horizontal Position" option.

Possible values are `left`, `center` and `right`. Default is `center`.


### 4. content-size <toujou-chip class="chip" chip-bg-color="warning">required</toujou-chip>
Allows us to set the header's content width in relation to the header element.

Corresponds to the "Headercontent Size" option.

Possible values are `full`, `half`, `third`,  and `quarter`. Default is `half`.


### 5. header-type
The `header-type` must be set to `header-type="slider"` so the different element inside the slider have the correct styles
