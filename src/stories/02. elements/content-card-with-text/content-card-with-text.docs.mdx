# Content card with text
The `content card with text` (either right or left) can be added on the "Combined elements" tab

This element is a combination of text elements and a `<toujou-content-card-grid>`

For more information please check the documentation for the `toujou-grid`, `toujou-content-card-grid` and `toujou-content-card` elements
