# &lt;toujou-icon&gt;
The `<toujou-icon>` is a simple custom element that allows us to render an icon on the page.


## Example
<toujou-icon class="icon" icon-name="star" icon-size="l" icon-color="primary"></toujou-icon>


## Markup example
```html
<toujou-icon class="icon" icon-name="account" icon-size="normal" icon-color="primary"></toujou-icon>
```

## Attributes
There are some attributes that allow us to customize the icon. we can change the icon image, size and color.


### icon-name <toujou-chip class="chip" chip-bg-color="warning">required</toujou-chip>
By setting the `icon-name` attribute we can choose which icon to show. For a list of all possible icons please see the story

### Icon color <toujou-chip class="chip" chip-bg-color="warning">required</toujou-chip>
With the `icon-color` attribute we can define the icon's color.

Valid values are:
- `primary`
- `primary-dark`
- `primary-light`
- `secondary`
- `secondary-dark`
- `secondary-light`
- `font`
- `font-dark`
- `font-light`,
- `background`,
- `success`,
- `warning`
- `error`

### Icon size <toujou-chip class="chip" chip-bg-color="warning">required</toujou-chip>
With the `icon-size` attribute we can define the icon's size.

Valid values are
- `xxs`,
- `xs`,
- `s`,
- `normal`,
- `ms`,
- `m`,
- `l`,
- `xl`,
- `xxl`
- `xxxl`
