# &lt;toujou-grid&gt;
The `<toujou-grid>` is a custom element to help us place content on the page. It provides a width and max-width and nothing else


## Markup example
```html
<toujou-grid grid-type="fullwidth">
    ...
</toujou-grid>
```


## Attributes
This element has a single attribute (`grid-type`) which defines how wide the grid is on the page


### grid-type [required]

| value | width                              | Example |
|---|---|:---|
| default | normal content width and max-width | `<toujou-grid grid-type="default">` |
| wide | wide content max-width | `<toujou-grid grid-type="wide">` |
| fullwidth | fullwidth, no paddings or margins | `<toujou-grid grid-type="fullwidth">` |
| ignore | `display: contents` so the grid element is ignored | `<toujou-grid grid-type="ignore">` |
