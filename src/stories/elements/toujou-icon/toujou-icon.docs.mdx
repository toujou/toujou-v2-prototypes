# &lt;toujou-icon&gt;
The `<toujou-icon>` is a simple custom element that allows us to render an icon on the page.


## Example
<toujou-icon class="icon icon--size-ms icon--color-font icon--account"></toujou-icon>


## Markup example
```html
<toujou-icon class="icon icon--size-ms icon--color-font icon--account"></toujou-icon>
```

## Attributes
There are some attributes that allow us to customize the icon. we can change the icon image, size and color.

<br/>

### icon-name [required]
By setting the `icon-name` attribute we can choose which icon to show. For a list of all possible icons please see the story

Examples:

| Name | icon-name | Example |
|---|---|:---:|
| account | `icon-name="account"`  | <toujou-icon class="icon icon--size-ms icon--color-font icon--account"></toujou-icon> |
| calendar-lines | `icon-name="calendar-lines"`  | <toujou-icon class="icon icon--size-ms icon--color-font icon--calendar-lines"></toujou-icon> |
| monument | `icon-name="monument"`  | <toujou-icon class="icon icon--size-ms icon--color-font icon--monument"></toujou-icon> |
| heart | `icon-name="heart"`  | <toujou-icon class="icon icon--size-ms icon--color-font icon--heart"></toujou-icon> |

<br/>
