# &lt;toujou-icon&gt;

The `<toujou-icon>` is a simple custom element that allows us to render an icon on the page.

## Example
<toujou-icon class="icon icon--size-ms icon--color-font icon--account"></toujou-icon>

## Code example
```html
<toujou-icon class="icon icon--size-ms icon--color-font icon--account"></toujou-icon>
```

## Properties
There are some properties you can set to customize the look of the icons

### Icon
By setting the `toujou-icon-name` property we can choose which icon to show

### Icon color

| Color  | Modifier   | Example |
|---|---|:---:|
| font color | `icon--color-font`  | <toujou-icon class="icon icon--size-ms icon--color-font icon--account"></toujou-icon> |
| primary color | `icon--color-primary` | <toujou-icon class="icon icon--size-ms icon--color-primary icon--account"></toujou-icon> |
| secondary color | `icon--color-secondary` | <toujou-icon class="icon icon--size-ms icon--color-secondary icon--account"></toujou-icon> |
